Distance in ULPs between values.

Computes the ULP distance between two values.

Semantic:

For any types T and U, any value a0 of type T and any value a1 of type U:

auto r = ulpdist(a0,a1);

is equivalent to:

auto r = ulpdist( static_cast<std::common_t<T,U>>(a0), static_cast<std::common_t<T,U>>(a1));

For any type T and any value of type T a0 and a1, then:

auto r = ulpdist(a0,a1);

is equivalent to:

  • if T is bool :
    auto r = a0==a1 ? 0. : 1.;
  • if T is an integral type:
    auto r = double( (a0>a1) ? a0-a1 : a1-a0 );
    i.e the number of different bits between a0 and a1
  • if T is a floating-point type, r is equal to the number of different bits between the normalized representation of a0 and a1 as defined by Golderg et al.
  • if T is a Container, r is a container containing the ULP distance between each pair of elements in a0 and a1
