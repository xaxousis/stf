Check for equality within a relative distance.

Evaluates A and B and checks if their respective value(s) are within a relative tolerance of X percent of each other. This test is performed by calling the default stf::reldist function or any ADL-accessible overload on each values from A and B.

Example:
STF_CASE("Relative distance between containers")
{
std::vector<float> a{1.f,1.1f,0.9f,1.3f};
std::vector<float> b{1.f,1.f,1.f,1.f};
std::vector<float> s{1.1f};
STF_RELATIVE_EQUAL(a , b , 25);
STF_RELATIVE_EQUAL(s , 1.f , 25);
STF_RELATIVE_EQUAL(1.f, s , 25);
}
