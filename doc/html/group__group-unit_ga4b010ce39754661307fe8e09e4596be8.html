Check for type equality on meta-lambda application.

For a given expression Expression and any type Lambda and Type, consider the test successful if and only if std::is_same<mpl::apply<Lambda,decltype(Expression)>::type,Type> evaluates to std::integral_constant<bool,true> .

Usage:
template<typename T> struct add_const_ref
{
using type = T const&;
};
STF_CASE( "Check that meta-lambda application can be tested" )
{
std::array<float,3> x;
STF_EXPR_TYPE( x , add_const_ref<boost::mpl::_>, (std::array<float,3> const&) );
}
