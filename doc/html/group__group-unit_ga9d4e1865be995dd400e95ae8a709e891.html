Starts a block defining a template test case.

The following block will contains user defined preparation, actual tests, expectations and clean-up for said test case parametrized by a template type named T and instantiated for each type in the TYPES sequence.

Such a test case is then defined as :

STF_CASE_TPL( "A template test", (float)(int) )
{
STF_PASS("Forced pass of " << sizeof(T) << " bytes.");
}

Note that, as for STF_CASE , empty tests are reported as automatic failures.
