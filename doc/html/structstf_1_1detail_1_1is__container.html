Description

template<typename T>
struct stf::detail::is_container< T >

Checks if a type fulfills the requirement of being accessible via a begin()/end()/size() interface.

Model:

Metafunction

Semantic:

For any type T:

using type = stf::detail::is_container<T>::type;

is equivalent to std::integral_constant<bool,true> if:

T t;
auto b = t.begin();
auto e = t.end();
auto s = t.size();

is well-formed. Otherwise, it i equivalent to std::integral_constant<bool,false> .
