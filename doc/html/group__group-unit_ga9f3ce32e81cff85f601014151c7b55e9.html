Check for type equality.

For a given expression Expression and a given type Type, consider the test successful if and only if std::is_same<decltype(Expression),Type> evaluates to std::integral_constant<bool,true> .

Usage:
STF_CASE( "Check that expression types can be tested for equality" )
{
double d;
boost::ignore_unused(d);
STF_EXPR_IS( ('c', &d + 5) , double* );
STF_EXPR_IS( std::move(d) , double&& );
STF_EXPR_IS( std::swap(d,d), void );
}
